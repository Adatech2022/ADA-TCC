VARIÁVEIS

Uma variável é um nome que definimos para armazenar dados. Usamos variáveis para manejar esses dados mais facilmente e para não os perder no processo.

É muito importante poder armazenar dados de forma temporária, essa é a função das variáveis, sempre que precisar armazenar algo, vamos usá-las, pois podemos inserir, remover ou alterar informações.

Regras para ser uma variável

Deve começar com uma letra;
Os próximos podem ser letras ou números;
Não podem utilizar nem um símbolo, exceto;
Não pode conter espaço em branco;
Não pode conter letras com acento;
Não pode ser uma palavra reservada;
